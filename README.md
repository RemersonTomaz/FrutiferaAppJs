
Conforme conversado previamente com o professor, foram desenvolvidas duas versões do projeto, e a abordagem foi aprovada antes da implementação.

📌 Versões Disponíveis
🔹 Versão 1 – Com Bootstrap

Utiliza o framework Bootstrap para estilização.

Layout responsivo facilitado pelo sistema de grid.

Componentes prontos como botões, modais e cards.

Desenvolvimento mais rápido e padronizado.

🔹 Versão 2 – Sem Bootstrap

Desenvolvida apenas com HTML, CSS e JavaScript puro.

Estilização personalizada.

Maior controle sobre o design e responsividade.

Implementação manual de layout, animações e componentes.
